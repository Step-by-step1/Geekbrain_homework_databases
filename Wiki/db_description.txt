База данных имитирует структуру данных сайта Википедия.

На сайте публикуются статьи (articles) которые могут создаваться как зарегистрированными пользователями (users),
так и не зарегистрированными (отмечаются как null в соответствующих колонках). Зарегистрированные пользователи 
делятся на обыкновенных пользователей и администраторов (admins). Статьи, созданные незарегистрированными
пользователями могут редактироваться кем угодно; зарегистрированные пользователи могут установить права редактирования
(access_types) только для других зарегистрированных пользователей или же для всех без исключения; администраторы могут
устанавливать также ограничение на редактирование только администраторами. Все редакции статей сохраняются в таблице 
history. Таблица media хранит медиа-файлы различных типов, указанных в таблице media_types. Для статей доступен форум (talk)
в котором могут участвовать только зарегистрированные пользователи.
Так же есть иной форум (teahouse_forum_talk) в котором пользователи задают темы (topics)
и дискутируют на них. Для обсуждения технических вопросов есть reference_desk в которой есть разделы (reference_desk_topics)
по которым люди задают вопросы (reference_desk_questions) и дают ответы (reference_desk_answers).
